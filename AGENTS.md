# Workspace routing index

- `Workforce-Docs/` — shared canonical Workforce documentation pack. Use this first for plans, current state, execution order, decisions, and progress before local repo notes.
- `WORKSPACE_PRIORITY_MAP.md` — quick priority map for planning, mission, and execution-order files.
- `workforce-dev/` — Workforce Developer Hub source and docs. Main local working copy for Dev Hub UI, scripts, packages, feature docs, planning queue, and copilot task builder.
- `workforce-backup/` — Legacy backend workspace and archived API/tenant/RBAC material.
- `workforce-showcase/` — Workforce Console showcase and frontend artefacts. Maps to PythonAnywhere’s `workforce_frontend_app` folder.
- `workforce-console/` — Console app/codebase used for the showcase frontend and related bundles.
- `Workforce-Console/` — Planning/docs hub for the console and frontend rollout.

If a task mentions PythonAnywhere frontend work, treat `workforce-showcase/` as the local mirror of `workforce_frontend_app`.
